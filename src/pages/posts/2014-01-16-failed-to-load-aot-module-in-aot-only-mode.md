Xamarin.iOS で Binding プロジェクトをプロジェクトに追加して実機で実行したら、起動時に落ちるようになってしまった。

ログには、

>Failed to load AOT module ‘/xxx/Hoge.dll.dylib' in aot-only mode.

というエラーが出ていた。

実機からアプリを削除して、
ソリューションを「再ビルド」したら直った。小ネタ過ぎる。
